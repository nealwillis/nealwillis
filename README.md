<h3>🚀 &nbsp;Featured Projects</h3>

**JobPilot** · Full-stack LLM application<br/>
A job-search copilot built on Next.js and TypeScript, with a Postgres and pgvector RAG pipeline, agentic tool-calling on the Anthropic Claude API, model routing for cost control, and a batch evaluation harness that scores output quality.<br/>
<sub>`Next.js` · `TypeScript` · `Claude API` · `PostgreSQL` · `pgvector` · `RAG`</sub>

**Chronic Disease Risk Predictor** · Machine learning<br/>
A classifier trained on CDC public-health data that predicts chronic disease risk, reaching 0.849 AUC.<br/>
<sub>`Python` · `scikit-learn` · `pandas`</sub>

**Biological Age Predictor** · Machine learning<br/>
A regression model that estimates biological age from health indicators, with a mean absolute error of 9.61 years.<br/>
<sub>`Python` · `scikit-learn`</sub>

**Network Intrusion Detection System** · Security ML<br/>
A detection model that flags malicious network traffic at a 99.98% detection rate.<br/>
<sub>`Python` · `scikit-learn`</sub>

**Paws Path** · Game-based learning platform<br/>
Tech lead on a game-based platform for criminal justice education at FVSU, built through the Valley Scholar Research Internship, including FERPA-compliant data pipelines.<br/>
<sub>`Python` · `Full-stack`</sub>
